# ProofLoop Web

ProofLoopのWebアプリケーションです。

## 技術スタック

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **ESLint**

## 開発の始め方

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## スクリプト

- `npm run dev` - 開発サーバーを起動
- `npm run build` - 本番ビルド
- `npm run start` - 本番サーバーを起動
- `npm run lint` - ESLintを実行
